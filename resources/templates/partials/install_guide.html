Install guide

{% include "templates/partials/slack_button.html" %}

Click on Slack button, select your team and pick a channel where you want to receive VersionEye notications and after successful authorization page you should see the success-page with small walktrough examples.

You need to have VersionEye API-token to access your projects on VersionEye. Copy token from API page and use /veye connect YOURTOKEN and you can now access your VersionEye data without leaving Slack.
