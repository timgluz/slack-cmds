Short introduction

VersionEye is a service that tracks a release of software libraries of various programming languages, so you can keep eye on hotfixes, breaking changes and conflicts of licenses without any hassle and it all happens on background.

Veye/Slack is a hip version of VersionEye command-line tool and most of commands in Veye/Slack are inspired by that CLI-tool.
This tool is open-source project and every feedback is very welcome.
