What next?

Just open your team's Slack and type /veye command.

VersionEye is now integrated with your team account. You can now use the "veye" command to search software packages, check statuses of your projects and re-check projects;

Additionally, the-veye command, you'll get regular notifications to the Slack channel; It wouldn't happen more than update frequency set on your project settings on the VersionEye.

Quick start

A command walk-through.

/veye search clojure finds software packages names as `clojure`
/veye connect YOUR-API-KEY saves VersionEye API key, so you could access details of dependencies and licenses on the VersionEye.
You can find your API key on your VersionEye settings page;
/veye list returns a list of your VersionEye projects.
/veye info PROJECT_UUID returns a list of outdated dependencies in your project.
/veye help returns a list of action.
